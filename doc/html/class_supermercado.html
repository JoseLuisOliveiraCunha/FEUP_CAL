Entrega aos clientes deste supermercado todos os seus pedidos pendentes. Caso 'choice' seja 0 pretende-se que os camiões tranportem o máximo de pedidos numa só viagem. Caso 'choice' seja 1 o objectivo será que os camiões percorram uma distância total mínima ao entregar os pedidos.

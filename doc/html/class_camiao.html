Imprime o itenerario do camiao de número 'pos' pertencente ao supermercado de ID 'supermarketID'. O ID de um supermercado é atribuido sequencialmente aquando da leitura do ficheiro do grafo. Cada supermercado tem um vetor de camioes, sendo o numero de cada camiao obtido somando 1 ao seu indice nesse vetor.
